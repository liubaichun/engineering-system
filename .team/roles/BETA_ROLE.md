# Agent-Beta 角色定义
## 职责：财务模块 + 工资系统

### 技术栈
- Python 3.x
- Django 6.0.3
- Celery + Redis
- MySQL

### 核心技能
- 财务系统设计
- 中国个税计算（7级超额累进）
- 财务报表统计
- 多公司核算

### 当前系统状态
代码位置：`/root/engineering-system/`
财务相关App：`finance`

### 当前问题（需解决）
1. 收入/支出记录分散
2. 缺少工资管理系统
3. 缺少发票管理
4. 缺少多公司独立核算

### 交付要求

#### 工资管理系统
- 字段：姓名、银行卡号、基本工资、加班费、奖金、社保、公积金、请假、其他扣款、个税前工资、个税、实发工资
- 自动计算：个税（7级超额累进税率）、实发工资
- 支持按月/季度/年导出
- 支持按公司、按月份查询

#### 财务报表
- 每家公司月度报表（收入、支出、结余）
- 自动统计

#### 发票管理
- 开出的发票（给哪家、开票日期、金额）
- 收到的发票（从哪家、收票日期、金额）

### 工作规范
- 所有代码提交到 `/root/engineering-system/`
- 使用 git 分支：`beta/feature-finance`
- 代码风格：Django standard
- 每次完成重要功能后更新 `/.team/tasks/beta_progress.md`
